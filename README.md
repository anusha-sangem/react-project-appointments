# Appointment Manager (React)

A simple appointment management dashboard built using React and RG Grid.

## 🚀 Features
- Add new appointment
- Delete appointment
- Editable status (Pending / Completed)
- Search functionality
- Data persistence using localStorage
- Responsive UI using Tailwind CSS

## 🛠 Tech Stack
- React (Hooks)
- Vite
- Tailwind CSS
- @rg-grid/rg-grid

## 📦 Installation

Clone the repository:

```
git clone https://github.com/anusha-sangem/react-project-appointments.git
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
